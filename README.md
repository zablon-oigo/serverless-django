
## Tech Stack

**Server:** Python, Django

## Run Locally

Clone the project

```bash
  git clone https://github.com/zablon-oigo/serverless-django.git
```

Go to the project directory

```bash
  cd serverless-django
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
 py manage.py runserver
```
